## Estructura del Proyecto

El portfolio está dividido en 9 componentes principales, cada uno en su propio archivo dentro de la carpeta `src/components`:

1. **Navbar.vue**: Barra de navegación principal.
2. **Presentacion.vue**: Presentación del grupo y del proyecto.
3. **Lenguajes.vue**: Lenguajes y aplicaciones aprendidos (Java, Python, Javascript, MySQL, Git, GitHub).
4. **Integrantes.vue**: Listado de los integrantes del grupo.
5. **InfoPersonal.vue**: Información personal de cada integrante (cada uno puede agregar su sección aquí).
6. **Contacto.vue**: Información de contacto del grupo.
7. **Footer.vue**: Pie de página.
8. **Galeria.vue**: Galería de imágenes (trabajos, clases, eventos, etc.).
9. **Proyectos.vue**: Proyectos grupales realizados.

## ¿Cómo colaborar?

Cada integrante debe subir su parte del portfolio siguiendo estos pasos:

1. Descargar el proyecto base desde el Drive o GitHub.
2. Agregar o modificar el componente que le corresponde en `src/components`.
3. Realizar un `git pull` para actualizar el repositorio local.
4. Arrastrar su archivo al proyecto y asegurarse de que no sobrescribe el trabajo de otros.
5. Hacer `git add`, `git commit` y `git push` para subir su parte.
